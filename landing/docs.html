IOS Marketing SDK






Introduction


Marketing SDK make application developer easily integrate BYPASSFIREWALL revenue stream to their apps.
Currently, we only support iOS applications.
In addition, we only call SDK in case of Wifi is used to prevent affect to celluar data bandwidth.




QuickStart


1. Setup


You'll need following before installing the SDK:
- Your API Key. Signup for a free account to get your API Key in Publisher Panel.
- iPhone (4 or newer) or iPad (3rd gen. or newer) with iOS 7+, to run your app.
- Mac computer with Xcode.


2. Install SDK


From your xcode project, add Marketing SDK library (download here) to your project

3. Init SDK


From your xcode project, add following line to AppDelegate.m
    @import MarketingSDK.h;

Then in application:didFinishLaunchingWithOptions: method add following line:
    [MarketingSDK init];

4. Configure SDK


Create data.conf in your root project with following format:
    {
        "token":"YOUR_API_KEY",
        "udid":"Device UDID",
        "pk":"App Bundle Id"
    }

YOUR_API_KEY is get from Account Manager in Publisher Panel. In addition, udid or pk is lacked then its value will be autodetected by SDK.

5. Test


Run your app with Wifi connection then check in Publish Panel statistic report to see a new event is reported.
Iy you rerun your app without IP change, you will see no new event is appeared.
Once again, note that, our system only run thought Wifi, You could test celluar and see that no event is reported in Publish Panel.
